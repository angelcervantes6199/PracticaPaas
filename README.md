# PracticaPaa

Objetivo:

Que el estudiante experimente el despliegue de una aplicación web en una plataforma PaaS, comprendiendo cómo se abstrae la gestión de la infraestructura y enfocándose en el código y la funcionalidad de la aplicación.

Resultados de aprendizaje:

· Comprensión clara del modelo PaaS.

· Aplicación en línea sin configurar servidores ni redes.

· Capacidad de documentar técnicamente un despliegue en la nube.

· Apreciación del ahorro de tiempo que ofrece una plataforma PaaS.
